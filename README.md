# nukecalc
Nuclear calculator for Factorio with support for Bob's mods and Space Explorarion mod.

# Requirements
### You need Python 3 to run nukecalc.
### You can download Python 3.10.4 for Windows [here.](https://www.python.org/ftp/python/3.10.4/python-3.10.4-amd64.exe)

# Usage
```bash
py nukecalc.py
```
